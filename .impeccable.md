## Design Context

### Users
Personal server administrator managing BT seeding operations. Technical user comfortable with terminals and dashboards. Uses the platform for long monitoring sessions, typically at night or in dim environments. The primary job is to monitor seeding health, adjust policies, and ensure optimal use of bandwidth/storage.

### Brand Personality
Refined · Precise · Calm — like a well-tuned instrument panel in a luxury cockpit.

### Aesthetic Direction
- **Palette**: Catppuccin Mocha — warm, soothing pastels on a deep base. NOT cold blue/purple neon.
- **Inspiration**: Linear's spatial clarity, Vercel's confidence, Raycast's density-without-clutter.
- **Anti-references**: Generic admin dashboards, glassmorphism-heavy AI output, Grafana's visual noise.
- **Theme**: Dark only (Catppuccin Mocha base #1e1e2e).
- **Typography**: Purposeful pairing. Display font with character, body font with clarity.
- **Layout**: Generous whitespace, clear section boundaries, NO card-within-card nesting, NO identical card grids.

### Design Principles
1. **Breathing room** — Space is a feature, not waste. Sections need generous padding and clear separation.
2. **Flat hierarchy** — No glassmorphism, no blur, no glow. Use Catppuccin surface levels for depth.
3. **Color with purpose** — Catppuccin accents only for semantic meaning (status, actions). Neutrals dominate.
4. **Typography-driven** — Let type size, weight, and color create hierarchy. Minimize decorative elements.
5. **Quiet confidence** — The interface should feel calm and in control, never busy or trying to impress.
